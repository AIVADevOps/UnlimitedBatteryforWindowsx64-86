WARUM DIESE SETTINGS FUNKTIONIEREN
================================================================================
Das Geheimnis: STAPM 8W + EDC 50A

- STAPM 8W = Langfristiger Durchschnittsverbrauch bleibt niedrig
- EDC 50A = Erlaubt kurze Mikrosekunden-Stromspitzen
- Diese Spikes kosten kaum Battery, machen aber UI instant-responsive
- Ergebnis: Tablet-Effizienz mit Desktop-Feeling

Niedriger EDC = CPU muss Strom "sanft" erhoehen = Micro-Lag
Hoher EDC = CPU kann sofort spiken = Butter-smooth 165Hz
